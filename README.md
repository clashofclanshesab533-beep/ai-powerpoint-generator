# AI PowerPoint Generator

## Project Overview
This project aims to automate the creation of PowerPoint presentations using AI technology. The tool can generate slides based on user-defined content, making it easier for users to create presentations quickly and efficiently.

## Features
- **Automated Slide Generation:** Create slides automatically based on the input text or keywords.
- **Customizable Templates:** Use various templates for different types of presentations.
- **AI-Powered Content Suggestions:** Get AI-generated suggestions for slide content and formatting.
- **User-Friendly Interface:** Easy to use interface that guides users through the process.

## Installation
To get started, clone the repository:
```bash
git clone https://github.com/user/repo.git
```

Navigate to the project directory:
```bash
cd ai-powerpoint-generator
```

Then install the required dependencies:
```bash
pip install -r requirements.txt
```

## Usage
Run the main script to start generating slides:
```bash
python main.py
```
